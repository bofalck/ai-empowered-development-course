BUTTON CONSOLIDATION
====================

BEFORE:
  28 separate button classes
  500+ lines of CSS
  18 lines per button average
  Scattered definitions
  Inconsistent theming

AFTER:
  8 base variants
  200 lines of CSS
  7 lines per button average
  Centralized definitions
  Complete theme coverage

REDUCTION: 71% fewer classes, 60% less CSS

KEY VARIANTS
============

.btn--primary       Blue buttons (Save, Submit, Record, OK)
.btn--secondary     Gray buttons (Cancel, Clear, Close)
.btn--icon          Icon toggles (Theme, Mode, Language)
.btn--success       Green buttons (Confirm, Apply, Accept)
.btn--danger        Red buttons (Delete, Sign Out)
.btn--warning       Amber buttons (Caution, Alert)
.btn--text          Minimal buttons (Tags, Links)
.btn--ghost         Link-style buttons

SIZE MODIFIERS
==============

.btn--sm     36px height
.btn--md     44px height (default)
.btn--lg     52px height
.btn--xl     60px height
